# TV Stream App

Projeto inicial para um app Android que lista canais de streaming com paywall e um backend simples de URL de streaming.

## Estrutura

- `backend/` - serviço Node.js para fornecer canais e URLs HLS de exemplo
- `android-app/` - esqueleto de app Android em Kotlin

## Próximos passos

1. Abra `tv-stream-app` no VS Code ou Android Studio.
2. Instale dependências do backend:
   - `cd backend`
   - `npm install`
3. Execute o backend:
   - `npm start`
4. Importe `android-app` no Android Studio para desenvolver o app mobile.
